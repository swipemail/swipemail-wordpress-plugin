SwipeMail WordPress Plugin - To Integrate SwipeMail Services with your WordPress

== Description ==
SwipeMail WordPress Plugin

We have released WordPress plugin. Currently, with the help of this plugin you can have a subscription form anywhere on your website. Any of your visitors can fill this form and their email address will get added to the list of your SwipeMail account.

In the near future, we may add more functionality like using our SMTP or API to send all your WordPress emails from your SwipeMail’s account and so forth.

== Installation ==

To install this plugin , follow these steps:

1.	Download and unzip the plugin
2.	Upload the entire swipemail/ directory to the /wp-content/plugins/ directory
3.	Activate the plugin through the Plugins menu in WordPress
4.  Go to "SwipeMail Pluggin Settings" -> Update your SwipeMail Username, Password, list ID and click on Save Changes button
5. Now edit your page and put this short-code [swipemail_subscribe_form]
6. Update your page and check in frontend !

For more information visit [http://swipemail.io/swipemail-wordpress-plugin/]
